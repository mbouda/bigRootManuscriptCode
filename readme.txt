bigRootManuscriptCode repository

Software

The repository contains source code that runs the analyses included in the manuscript "A big root approximation of site-scale vegetation water uptake", under consideration at Journal of Advances in Modelling Earth Systems.

All analyses are run from the script driveAll.m, all other source files are functions executed from this file. Furhter notes on code structure and execution are also found in that file as comments.
